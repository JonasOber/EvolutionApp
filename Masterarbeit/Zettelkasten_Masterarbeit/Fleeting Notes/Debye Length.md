# Debye Length
Created: 2021-12-1411:14
The debye length in semiconductors:
$ \begin{math}
\lambda_D = \sqrt{\frac{\epsilon_0\,\epsilon_r\,U_T}{e\,\rho}}
\end{math}$
Data for perovskites:
|           | Perovskite     |
| --------- | -------------- |
| epsilon_0 | 8.85e-12 As/Vm |
| epsilon_r | 24.1           |
| U_T       | 25e-3   V      |
| e         | 1.6e-19 C      |
| n_i       | 4.09e-4 cm^-3   |
| --------- | -----------    |
| lambda_D  | sehr groß      |
